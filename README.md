# aardio-paddleOCR
aardio中调用paddleOCR的api实现文字识别

这里是简单的调用在线api实现.
paddleOCR脱机版貌似要用到vs和Python , 我就没测试了.

类似的在线ocr, 目前有很多, 百度的ocr , 阿里的,搜狗的, [aardio软件](http://www.aardio.com)中已经写好了相关的调用库, 可以直接使用.
本文首发: https://www.htmlayout.cn/t/408
